# Scientific Calculator

## Overview

Scientific Calculator is a menu-driven application developed in C that provides a wide range of mathematical and scientific operations. The calculator supports arithmetic calculations, trigonometric functions, number system conversions, matrix operations, and complex number computations.

## Features

### Arithmetic Operations

* Addition
* Subtraction
* Multiplication
* Division

### Scientific Functions

* Square Root
* X^Y
* X²
* X³
* 1/X
* X^(1/Y)
* X^(Y1/Y2)
* 10^X
* Factorial
* Percentage
* Logarithm (log10)
* Modulus

### Trigonometric Functions

* Sin(X)
* Cos(X)
* Tan(X)
* Sec(X)
* Cosec(X)
* Cot(X)

### Number System Conversions

* Decimal ↔ Binary
* Decimal ↔ Octal
* Decimal ↔ Hexadecimal
* Binary ↔ Decimal
* Binary ↔ Octal
* Binary ↔ Hexadecimal
* Octal ↔ Decimal
* Octal ↔ Binary
* Octal ↔ Hexadecimal
* Hexadecimal ↔ Decimal
* Hexadecimal ↔ Octal
* Hexadecimal ↔ Binary

### Matrix Operations

* Matrix Addition
* Matrix Subtraction

### Complex Number Operations

* Complex Number Addition
* Complex Number Subtraction
* Complex Number Multiplication
* Complex Number Division

### Additional Features

* Menu-Driven Interface
* Calculation History Storage
* File Handling with Results Export
* Structured Flowchart Documentation

## Technology Stack

* C Programming Language
* Standard C Libraries
* Math Library (math.h)
* File Handling
* Structured Programming

## Project Architecture

```text
Main Menu
```
